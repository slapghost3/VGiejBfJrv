# 图书管理系统 python26

## 项目概述

本系统是基于Python 3.7和Django框架，使用MySQL数据库开发的图书管理系统。系统提供了一系列功能，方便用户进行图书的入库、查询、借阅与归还等操作。

## 技术栈

- **后端语言与框架**：Python 3.7，Django
- **数据库**：MySQL 5.7+（推荐使用MySQL 8.0）

## 功能模块

- **用户登录**：用户可以通过登录功能进入系统。
- **图书入库**：支持单本及批量图书信息的录入。
- **图书查询**：用户可以检索图书信息。
- **借书证管理**：管理借书证信息，包括发放与回收。
- **借书与还书**：实现图书的借阅与归还操作。
- **借书记录查询**：可以查看图书的借阅记录。

## 系统角色

- **管理员**：负责图书的入库、管理和借书证的发放。
- **普通用户**：可以进行图书查询、借阅和归还。

## 运行环境

- **Python版本**：3.7
- **数据库版本**：MySQL 5.7+（推荐使用8.0）

## 部署步骤

1. 确保Python 3.7和MySQL 5.7+环境配置正确。
2. 按照提供的文档逐步操作，通过Django的migration文件进行数据表迁移。
3. 启动Django应用。
